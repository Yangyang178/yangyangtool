# 百宝工具箱 - 微信小程序

## 项目概述
这是一个高频实用的工具集小程序，包含 20 个常用工具，涵盖计算转换、文本处理、生活助手、日期时间和开发调试等类别。

## 功能特性

### ✅ 已实现功能
- **首页工具展示**：2列网格布局，20个高频实用工具
- **实时搜索**：支持关键词搜索，即时筛选工具
- **分类标签**：6个分类快速切换（全部/计算转换/文本处理/生活助手/日期时间/开发调试）
- **收藏功能**：可收藏常用工具，数据本地持久化存储
- **最近使用记录**：自动记录使用历史，显示使用时间
- **下拉刷新**：支持下拉刷新功能
- **触感反馈**：关键操作提供震动反馈
- **问候语显示**：根据时间显示不同问候语和日期
- **空状态处理**：无结果时显示友好提示

### 🎨 设计特点
- 柔和清爽的蓝色主题（#3B82F6）
- 卡片式设计，圆角 20px
- 渐变色图标背景
- 流畅的点击动画效果
- 完美适配手机竖屏（最大宽度 500px）

## 使用前准备

### ⚠️ 重要：底部导航栏图标配置

微信小程序的 tabBar 需要使用 **PNG 格式** 的图标文件。我已经为你创建了 SVG 格式的图标源文件，你需要将其转换为 PNG：

#### 方法一：在线转换（推荐）
1. 访问 https://convertio.co/svg-png/ 或类似网站
2. 上传 `images/` 目录下的 6 个 SVG 文件
3. 设置尺寸为 **81x81 像素**
4. 下载并重命名为：
   - `tab-tools.png` (未选中)
   - `tab-tools-active.png` (选中)
   - `tab-fav.png` (未选中)
   - `tab-fav-active.png` (选中)
   - `tab-recent.png` (未选中)
   - `tab-recent-active.png` (选中)

#### 方法二：使用设计工具
1. 用 Figma/Sketch/Illustrator 打开 SVG 文件
2. 导出为 81x81 像素的 PNG（透明背景）
3. 放入 `images/` 目录

#### 方法三：使用 Node.js 脚本
```bash
# 安装依赖
npm install sharp

# 运行转换脚本（需要创建）
node convert-icons.js
```

## 本地开发

### 环境要求
- 微信开发者工具（最新版本）
- Node.js (可选，用于后续功能扩展)

### 启动步骤
1. 打开微信开发者工具
2. 导入项目，选择项目根目录
3. 确保已将 SVG 图标转换为 PNG 并放入 `images/` 目录
4. 点击"编译"运行项目
5. 在模拟器中预览效果

### 数据存储
所有数据使用 `wx.storage` 本地存储：
- `favorites`: 收藏的工具ID数组
- `recentTools`: 最近使用的工具记录（最多20条）

## 技术栈

- **框架**: 微信小程序原生框架
- **渲染**: Skyline 渲染引擎（性能优化）
- **组件**: Glass-easel 组件框架
- **样式**: WXSS（类似CSS，支持部分新特性）
- **布局**: Flexbox + Grid

## 核心代码说明

### 首页 (index.js)
- `filterTools()`: 核心筛选逻辑，支持分类+搜索双重过滤
- `toggleFavorite()`: 切换收藏状态，同步到本地存储
- `saveRecentTool()`: 保存使用记录，自动去重并限制数量
- `updateGreeting()`: 动态更新问候语和日期

### 收藏夹 (favorites.js)
- `loadFavorites()`: 从本地存储加载收藏列表
- `removeFavorite()`: 取消收藏并刷新列表

### 最近使用 (recent.js)
- `loadRecentTools()`: 加载并格式化时间戳为可读格式
- `formatTime()`: 智能时间格式化（刚刚/X分钟前/X小时前...）

## 交互细节

1. **点击反馈**: 卡片按下时 scale(0.98) + 透明度变化
2. **搜索体验**: 无延迟防抖，输入即过滤
3. **收藏动效**: 星星颜色从灰色变为金色
4. **触感反馈**: 关键操作调用 `wx.vibrateShort`
5. **下拉刷新**: 1秒后完成刷新并提示

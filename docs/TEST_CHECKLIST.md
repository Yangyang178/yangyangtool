## 🛠️ 常见问题排查

### Q1: 测试发现Bug怎么记录？

**使用Issue模板**:
```markdown
**Bug标题**: [组件名] [简短描述]

3. 观察到xxx现象
**预期行为**: 应该xxx
**实际行为**: 实际xxx
**环境信息**: iOS 14.2 / 微信 8.0.48
**截图**: [附图]
**日志**: [控制台报错]
```

### Q2: 如何判断是代码问题还是环境问题？

**排查步骤**:
1. 在另一台设备上复现 → 若不复现，可能是设备特定问题
2. 切换网络（WiFi/4G）→ 若恢复正常，可能是网络问题
3. 清除缓存重启小程序 → 若解决，可能是缓存问题
4. 回退到上一版本 → 若消失，是本次引入的Bug

### Q3: 性能测试数据不准确？

**确保测试条件一致**:
- 关闭开发者工具的调试模式
- 使用真机而非模拟器
- 多次取平均值（至少3次）
- 在同一网络环境下测试
